# pi stuff

I'm experimenting with Pi and extensions. Some are copied from https://github.com/mitsuhiko/agent-stuff and some are my own tests.

## Extensions

- `context.ts` — Shows loaded context.
- `files.ts` — Browse and open files.
- `session-breakdown.ts` — Shows session stats.
- `test.ts` — Scratch extension experiments.

## Skills

- `grill` — Asks clarifying questions.
- `ts-setup` — Sets up TypeScript formatting and checks.
